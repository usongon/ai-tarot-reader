# 神秘塔罗 - 现代化前端重构

## 🎨 重构亮点

### 🚀 技术升级
- **构建工具**: 从 Create React App 迁移至 Vite，开发速度提升 50%+
- **现代化框架**: React 18 + 最新生态
- **样式系统**: 自定义 CSS 架构，替代 Tailwind CSS 以获得更好的控制
- **动画引擎**: Framer Motion 实现流畅交互动画

### 🎯 设计革新
- **神秘优雅主题**: 深色渐变背景 (#1a1a2e → #16213e → #0f3460)
- **玻璃拟态效果**: 现代化的毛玻璃界面设计
- **流体动画**: 页面切换、悬停效果、加载状态的平滑过渡
- **响应式布局**: 完美适配桌面、平板、手机设备

### 🧩 组件架构
```
src/
├── components/
│   ├── layout/          # 布局组件
│   │   ├── Header.jsx   # 导航头部
│   │   ├── Footer.jsx   # 页脚
│   │   └── Layout.jsx   # 主布局容器
│   ├── pages/           # 页面组件
│   │   ├── HomePage.jsx         # 首页
│   │   ├── SpreadSelectionPage.jsx  # 牌阵选择
│   │   ├── TopicSelectionPage.jsx   # 话题选择
│   │   ├── DrawingPage.jsx          # 抽牌界面
│   │   └── HistoryPage.jsx          # 历史记录
│   ├── ui/              # 基础UI组件
│   │   └── Button.jsx   # 按钮组件
│   └── cards/           # 卡牌相关组件
├── contexts/            # 状态管理
│   └── TarotContext.jsx # 塔罗占卜上下文
├── hooks/               # 自定义Hooks
└── utils/               # 工具函数
```

## 🌟 核心特性

### 1. 视觉设计
- **色彩体系**: 紫色系主色调，搭配渐变和光影效果
- **字体**: Inter 字体家族，确保优秀的可读性
- **图标**: Heroicons 24 系列，简洁现代
- **装饰元素**: 浮动粒子、背景光晕等细节增强

### 2. 用户体验
- **流畅动画**: 页面切换、组件交互都有精心设计的动画
- **直观导航**: 清晰的步骤引导和返回机制
- **即时反馈**: 按钮点击、悬停状态的视觉反馈
- **加载状态**: 优雅的加载指示器

### 3. 响应式设计
- **移动优先**: 针对小屏幕优化的布局
- **弹性网格**: 自适应不同屏幕尺寸的卡片布局
- **触摸友好**: 适合触屏操作的按钮大小和间距

## 🛠️ 开发指南

### 启动开发服务器
```bash
cd frontend/temp-app
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产构建
```bash
npm run preview
```

- 基础间距: 1rem (16px)

- 大间距: 2rem (32px)
